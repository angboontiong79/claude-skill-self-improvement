# Self-Improving Claude Code v2.1

A modular, extensible self-improvement system for Claude Code.

## Features

- Two-tier system (gate/periodic analysis)
- Telemetry tracking
- Semantic friction analysis
- Severity levels
- State versioning with migration
- Modular plugin architecture

## Installation

```bash
git clone https://github.com/angboontiong79/claude-skill-self-improvement.git ~/.claude/skills/self-improvement
```
